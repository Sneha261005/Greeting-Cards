# Greeting Cards Website

A beautiful, modern greeting cards website built with React.js, HTML, CSS, and JavaScript. Features a responsive design with card customization options for various occasions.

## Features

- **Modern UI Design**: Beautiful gradient backgrounds and smooth animations
- **Card Categories**: Browse cards by occasion (Birthday, Wedding, Anniversary, Thank You, Congratulations, Get Well)
- **Card Customization**: Personalize cards with custom messages, recipient/sender names, and color themes
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Interactive Gallery**: Hover effects and smooth transitions
- **Modal Interface**: Clean customization experience

## Technologies Used

- **React.js**: Component-based frontend framework
- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with animations and transitions
- **JavaScript**: Interactive functionality and state management
- **Font Awesome**: Icon library
- **Google Fonts**: Typography (Poppins and Dancing Script)

## Getting Started

1. **Open the Website**: Simply open `index.html` in your web browser

2. **Browse Cards**: 
   - View all cards or filter by category
   - Click on any card to customize it

3. **Customize Cards**:
   - Add recipient and sender names
   - Write custom messages
   - Choose from 6 different color themes
   - Preview changes in real-time

4. **Card Categories**:
   - Birthday Cards
   - Wedding Cards
   - Anniversary Cards
   - Thank You Cards
   - Congratulations Cards
   - Get Well Soon Cards
